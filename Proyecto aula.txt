<h2 class="texto j">Introducción</h2>
<h3 class="texto j">En el marco del proyecto aula "Escuela Verde con Camino a la Sustentabilidad", la asignatura de Computación Básica II tuvo un rol fundamental como plataforma integradora de todos los contenidos generados por las distintas materias participantes. Nuestra misión fue desarrollar una herramienta tecnológica que permitiera documentar, organizar y compartir de manera accesible y atractiva los diversos aportes al proyecto.</h3>

<h3 class="texto j">La página web elaborada representa no solo un producto final, sino también un proceso de aprendizaje donde aplicamos conocimientos fundamentales de desarrollo web utilizando HTML5 y CSS básico. Este proyecto nos permitió comprender cómo la tecnología puede servir como puente entre diferentes disciplinas académicas y como herramienta para la difusión de iniciativas sustentables.</h3>

<h2 class="texto j">Objetivos del Proyecto</h2>
<h3 class="texto j">Los objetivos específicos de nuestra participación fueron:</h3>
<ul class="texto j">
    <li>Diseñar y desarrollar una página web responsive que integrara todos los contenidos del proyecto aula</li>
    <li>Aplicar los conceptos de estructura semántica HTML aprendidos en clase</li>
    <li>Implementar estilos CSS básicos para mejorar la presentación visual</li>
    <li>Organizar la información de manera lógica y accesible</li>
    <li>Crear un sistema de navegación interna entre secciones</li>
    <li>Optimizar imágenes y recursos multimedia para web</li>
    <li>Trabajar colaborativamente usando metodologías básicas de desarrollo web</li>
</ul>

<h2 class="texto j">Proceso de Desarrollo</h2>
<h3 class="texto j">El desarrollo del sitio web siguió una metodología por etapas:</h3>

<h3 class="texto j">1. Análisis de Requerimientos</h3>
<ul class="texto j">
    <li>Reunión con representantes de otras materias para entender sus necesidades</li>
    <li>Definición de la estructura general del sitio</li>
    <li>Identificación de los tipos de contenido a incluir</li>
</ul>

<h3 class="texto j">2. Diseño de la Estructura</h3>
<ul class="texto j">
    <li>Creación de un diagrama de flujo del sitio</li>
    <li>Definición de la jerarquía de información</li>
    <li>Diseño del sistema de navegación</li>
    <li>Selección de paleta de colores acorde al tema ecológico</li>
</ul>

<h3 class="texto j">3. Implementación Técnica</h3>
<ul class="texto j">
    <li>Maquetación HTML5 semántica con etiquetas como header, section, article, footer</li>
    <li>Uso de divs para organización del layout</li>
    <li>Implementación de estilos CSS para:
        <ul>
            <li>Tipografía (Google Fonts)</li>
            <li>Colores y fondos</li>
            <li>Diseño de cajas (márgenes, padding, bordes)</li>
            <li>Disposición flexible (Flexbox)</li>
        </ul>
    </li>
    <li>Inserción y optimización de imágenes</li>
    <li>Creación de reglas CSS para diferentes tamaños de pantalla</li>
</ul>

<h3 class="texto j">4. Integración de Contenidos</h3>
<ul class="texto j">
    <li>Recopilación y organización de textos proporcionados por otras materias</li>
    <li>Formateo consistente de encabezados y párrafos</li>
    <li>Creación de listas ordenadas y no ordenadas</li>
    <li>Implementación de enlaces internos y externos</li>
</ul>

<h3 class="texto j">5. Pruebas y Ajustes</h3>
<ul class="texto j">
    <li>Validación del código HTML</li>
    <li>Pruebas de visualización en diferentes navegadores</li>
    <li>Ajustes de responsividad</li>
    <li>Corrección de errores reportados por usuarios de prueba</li>
</ul>

<h2 class="texto j">Desafíos Técnicos y Soluciones</h2>
<h3 class="texto j">Durante el desarrollo enfrentamos varios desafíos técnicos:</h3>

<h3 class="texto j">1. Organización del Layout</h3>
<h3 class="texto j"><strong>Problema:</strong> Crear una estructura de dos columnas donde la barra lateral permaneciera fija mientras el contenido principal fuera desplazable.</h3>
<h3 class="texto j"><strong>Solución:</strong> Implementamos un contenedor flex con dirección row-reverse y asignamos un ancho fijo a la barra lateral mientras que el contenido principal ocupaba el espacio restante con flex: 1.</h3>

<h3 class="texto j">2. Responsividad</h3>
<h3 class="texto j"><strong>Problema:</strong> Hacer que el sitio se visualizara correctamente en diferentes tamaños de pantalla.</h3>
<h3 class="texto j"><strong>Solución:</strong> Usamos unidades relativas como porcentajes para anchos y em/rem para tamaños de fuente, además de media queries básicas para ajustar el diseño en pantallas pequeñas.</h3>

<h3 class="texto j">3. Integración de Contenido Diverso</h3>
<h3 class="texto j"><strong>Problema:</strong> Cada materia proporcionó información en formatos diferentes que necesitaban ser estandarizados.</h3>
<h3 class="texto j"><strong>Solución:</strong> Creamos plantillas HTML consistentes para cada sección y normalizamos los estilos CSS para mantener coherencia visual.</h3>
